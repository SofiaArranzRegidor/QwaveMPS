# Installation

## Requirements

- Python >=3.6
- Required packages: numpy, scipy, ncon.

## Steps

1. Clone the repository:
    ```bash
    git clone https://github.com/SofiaArranzRegidor/QwaveMPS.git
    ```
2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3. (Any additional steps, e.g. setting environment variables)
